PROJECT OVERVIEW

Vision-Based Drone Control

We developed an autonomous drone system that tracks and follows ArUco markers using computer vision, ROS2, and custom control algorithms. The system achieves real-time marker detection and smooth trajectory planning.

SYSTEM ARCHITECTURE

Three-Stage Pipeline

Our system uses a modular architecture with three main components: Perception, Planning, and Control

Perception

ArUco Detection Node

  • Subscribe: /arducam/image_raw
  • Process: cv2.aruco detection
  • Detect 4 corners + center
  • Publish: /aruco_detector/marker_pixel

Planning

Velocity Computation

  • Calculate pixel x/y error
  • Error = marker_center - image_center
  • Compute v_y, v_z (v_x = 0)
  • Publish: /desired_pose
üéÆ

Control

RC Command Mapping

  • Map velocity ‚Üí RC commands
  • [roll, pitch, throttle, yaw]
  • Range: 1000-2000 (neutral: 1500)
  • Publish: /rc/override
